## SECTION 1: Data Collection and Security

### Question: "Does your app collect or share any of the required user data types?"

**Answer: NO — with the following exception:**

The app only optionally transmits **chat message text** to Google Gemini API and Supabase Edge Functions when the AI Butler feature is used. This text contains no personal identifiers (no name, email, phone, location, or device ID is included).

No personal data is collected, stored in the cloud, or shared with any data broker or advertiser.

---

### Question: "Is all of the user data collected by your app encrypted in transit?"

**Answer: YES**

- Chat messages to Gemini API use **HTTPS/TLS 1.2+** (googleapis.com)
- Chat relay via Supabase uses **HTTPS/TLS** (supabase.co)
- Local PC communication uses **Bearer token authentication** over LAN, with optional self-signed **TLS/HTTPS** (generated by butler_server.py)

---

### Question: "Do you provide a way for users to request that their data is deleted?"

**Answer: YES**

Users can delete all their data by:
1. **Uninstalling the app** — Android deletes all AsyncStorage data automatically
2. **Clear Knowledge Base** — Settings tab → Clear Knowledge Base button
3. **Clear Chat** — Butler AI tab → trash icon → "Clear Chat"
4. **Disconnect** — Connect tab → DISCONNECT removes server credentials
